<objective>
## Goal
Add Stripe billing with two tiers (Essentials $199/mo, Growth $499/mo) and a client onboarding wizard that captures shop info and creates the shop + membership records. After this plan, new customers can sign up, onboard their shop, and subscribe to a billing tier.

## Output
- Stripe Checkout integration for subscription creation
- Customer portal for managing subscriptions
- Webhook handler for subscription lifecycle events
- Billing schema migration (subscriptions table)
- Onboarding wizard (shop name, address, website URL, auto-detect)
- Pricing page showing both tiers
</objective>

```gherkin

When they visit /dashboard/onboarding
Then they see a multi-step wizard: shop name, address, website URL

And they are redirected to the dashboard
```

## AC-3: Webhook and billing schema
```gherkin

And the shop's billing tier is reflected in the dashboard
```
