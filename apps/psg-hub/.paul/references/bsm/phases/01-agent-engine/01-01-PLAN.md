<task type="auto">
  <name>Task 2: Create the SEO auditor SKILL.md</name>
  <files>~/.claude/skills/bsm-seo-auditor/SKILL.md</files>
  <action>
    Use /skillsmith to create the skill. The SKILL.md must define:

    **Persona:**
    - SEO specialist with deep collision repair vertical knowledge
    - Understands local search intent (claimants searching after accidents, people choosing shops)
    - Knows which technical SEO issues matter most for small business service sites
    - Speaks in PSG brand voice (calm, direct, specific, no filler)

    **Input:**
    - Shop profile JSON (website_url required, competitors optional)
    - Optional: specific audit focus (technical, content gaps, competitor comparison)

    **Tools the skill can use:**
    - Firecrawl MCP (site crawling and page analysis)
    - WebFetch (checking specific URLs, robots.txt, sitemap.xml)
    - WebSearch (competitor research, SERP position checking)
    - Grep/Read (analyzing scraped content)
    - Bash (running lighthouse or similar CLI tools if available)

    **Output schema (structured markdown):**
    ```
    # SEO Audit: {shop_name}
    ## Technical findings
    | Finding | Severity | Page/URL | Recommendation |
    ## Keyword analysis
    | Keyword | Monthly volume | Current rank | Competitor rank | Gap |
    ## Content coverage
    | Topic/Service | Has page? | Quality score | Recommendation |
    ## Competitor comparison
    | Metric | {shop} | {comp1} | {comp2} | {comp3} |
    ## Prioritized recommendations
    1. [High] ...
    2. [Medium] ...
    3. [Low] ...
    ```

    **Decision logic:**
    - Missing service pages for offered services = high priority
    - No Google Business Profile optimization = high priority
    - Missing schema markup (LocalBusiness, AutoRepair) = medium priority
    - Thin content (< 300 words on service pages) = medium priority
    - Missing meta descriptions = medium priority
    - Image alt text missing = low priority
    - Competitor outranking on primary keywords = high priority (flag for market researcher)

    **Inter-agent interface:**
    - Can be invoked by content writer to get keyword targets for a specific topic
    - Produces content gap findings that feed into market researcher briefs
    - Accepts queries like "what keywords should a blog post about hail damage repair target in Lincoln NE?"

    Avoid: Generic SEO advice that applies to any industry. Every recommendation must reference collision repair specifics (services, certifications, insurance processes, customer journey).
  </action>
  <verify>
    - SKILL.md exists at ~/.claude/skills/bsm-seo-auditor/SKILL.md
    - YAML frontmatter includes name, description, and tools
    - Skill defines input format, output schema, and decision logic
    - Collision repair domain knowledge is embedded (not generic SEO)
  </verify>
  <done>AC-1 satisfied: skill file created and loadable by Claude Code</done>
