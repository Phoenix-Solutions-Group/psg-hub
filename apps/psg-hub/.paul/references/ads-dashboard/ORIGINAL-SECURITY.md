# Security Policy

## Reporting a Vulnerability

Email security issues to **security@phoenixsolutionsgroup.net**.
Do not file public GitHub issues for security reports.

We will acknowledge receipt within **2 business days** and provide a
remediation timeline within **5 business days**.

## Supported Versions

Only the current production deployment is supported. We do not backport
fixes to prior versions.

## Scope

This policy covers:

- The Ads Dashboard web application
- Its Supabase data layer
- The Python sync job that populates the data layer
- CI/CD configuration in this repository

Out of scope: Google Ads platform itself, Vercel platform, Supabase platform.
Report vulnerabilities in those to their respective vendors.
