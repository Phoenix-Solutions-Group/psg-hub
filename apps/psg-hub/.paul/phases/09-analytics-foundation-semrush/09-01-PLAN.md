## Goal
Stand up the source-agnostic analytics data model (one migration, applied LOCAL-only) and the PSG-branded Recharts chart primitives — the reusable foundation every v0.3 source (SEMrush, Google Ads, GA4, GSC) and the Phase-12 report build on.

## Purpose
v0.3's load-bearing decision is the data model: it is a shared-prod schema that phases 10/11/12 inherit. Getting it source-agnostic now means adding each later source is "new `source` value + adapter," never a schema change. The chart primitives are the brand-conform viz layer (Recharts-direct, no Tremor/Tailwind-4 coupling) that the dashboard shell (09-02) and panels (09-03) consume.
