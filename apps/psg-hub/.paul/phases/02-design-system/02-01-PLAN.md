<objective>
## Goal
Vendor the PSG design system as a git submodule at `packages/ui/psg-brand/`, wire its brand fonts (Gotham + Didact Gothic) into psg-hub via `next/font/local`, and replace the inherited BSM "Clarity Teal" theme with the real PSG brand tokens (midnight navy + phoenix ember) in psg-hub's Tailwind 4 `@theme`. After this plan, psg-hub renders in the PSG brand and the design-system repo is the single source of brand truth.

## Purpose
Phase 1 consolidated the workspace but psg-hub still carries BSM's wrong brand: teal `#0D9488` + gold + Geist fonts. PSG's actual brand is midnight navy `#1E3A52`, phoenix ember `#B8483E`, dark-ash `#4B5058`, paper neutrals, Gotham headings, Didact Gothic body. This plan kills the teal and binds psg-hub to the upstream brand repo so every later UI milestone inherits correct tokens. Directly serves the PROJECT constraint "strict conform to PSG design system (no extensions)" and the tracked metric "Brand conformance audit per UI milestone | Pass."

## Output
- `.gitmodules` + checked-out submodule at `packages/ui/psg-brand/`
- `apps/psg-hub/src/lib/fonts.ts` (next/font/local for Gotham + Didact Gothic)
- Rewritten `:root` / `.dark` token values + font bindings in `apps/psg-hub/src/app/globals.css`
- `apps/psg-hub/src/app/layout.tsx` using brand font variables (Geist removed)
- `apps/psg-hub/src/styles/tokens.css` deleted
- `.paul/phases/02-design-system/02-01-SUMMARY.md`
</objective>
