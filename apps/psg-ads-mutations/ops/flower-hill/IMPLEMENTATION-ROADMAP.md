## Phase 4: Optimize (Weeks 7-10)

### Weekly Routine
| Day | Action |
|-----|--------|
| Monday | Pull week-over-week performance; adjust bids on underperformers |
| Wednesday | Search terms audit — add negatives, flag new keyword opportunities |
| Friday | Creative performance review — pause CTR outliers, brief replacements |

### Optimization Actions (trigger-based)
| Trigger | Action |
|---------|--------|
| Campaign reaches 15+ conversions/month | Switch from Maximize Clicks → Maximize Conversions |
| Campaign reaches 30+ conversions/month | Switch to Target CPA (set at 20% above current CPA) |
| Ad CTR < 1.5% after $100 spend | Pause ad, replace with new variant |
| Ad set frequency > 3.5 | Refresh creative or expand audience |
| Keyword CPL > $200 for 2 weeks | Pause keyword or reduce bid |
| LSA lead invalid | Dispute in LSA dashboard within 30 days |

### Month 2 Deliverables
- [ ] Performance report: CPL by platform, call volume by location
- [ ] Creative refresh: replace 2 underperforming ads
- [ ] Audience expansion test: lookalike audiences (if pixel has 500+ events)
- [ ] Exotic tier assessment: are luxury keywords generating calls?

## Phase 5: Scale (Weeks 11-12 and Beyond)

### Unlock YouTube (Month 3)
- [ ] 60-90 second brand video produced (in-shop, exotic car focus)
- [ ] YouTube campaign created — Huntington + North Shore focus
- [ ] Targeting: luxury auto interests + custom intent (searched for exotic repair)
- [ ] Budget: $200-$400/month trial (from testing allocation)

### Bing/Microsoft Test (Month 3)
- [ ] Import Google campaigns to Microsoft Ads
- [ ] Adjust bids down 30% (Bing CPCs are lower)
- [ ] Run 60 days — compare CPL to Google; keep if CPL < $120

### Quarterly Review (Month 3, Month 6, Month 12)
- Full budget reallocation based on actual CPL data
- Location-level performance review — is Huntington on par with others?
- Exotic certification ROI: what ticket value came from luxury ad traffic?
- Competitive landscape update (new competitors, keyword shifts)
